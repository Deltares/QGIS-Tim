This QGIS plugin serves to easily create model input timflow analytic
element models:

* [timflow](https://github.com/timflow-org/timflow)

Analytic element input is typically a collection of points, lines, and polygons.
This plugin represent a single model with a single GeoPackage. The different
elements are represented as layers in the GeoPackage, and also as ordinary
vector layers in QGIS.

To run the model with timflow, this plugin relies on a server program called `gistim`.

Find the repository and issue tracker here:
https://github.com/Deltares/QGIS-Tim
